#### 提交命令
1. 查看状态
>git status
2. 文件修改路径
>git add .
3. 添加注释
>git commit -m '注释内容'
4. 提交代码
>git push -u origin master

#### 更新命令
1. 查看状态
>git status
2. 查看当前分支
>git branch
3. 如果为本地分支，切换到远程分支
>git checkout remote branch
4. 更新
>git pull

#### 强制更新
>git fetch --all && git reset --hard origin/master && git pull

#### 运行命令
>java -Dfile.encoding=utf-8 -jar spring-boot-demo.jar
